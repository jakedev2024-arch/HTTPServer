# HTTPServer

HTTPServer provides HTTP server functionality for iOS applications.

## Requirements

- iOS 13.0+
- Swift 5.7+

## Installation

### Swift Package Manager

Add the following to your `Package.swift` file:

```swift
dependencies: [
    .package(url: "https://github.com/yourusername/HTTPServer.git", from: "0.1.0")
]
```

## Usage

```swift
import HTTPServer
```
